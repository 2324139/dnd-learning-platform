# 🌍 知識大陸 - 完整世界觀設定

## 第一卷 · 世界基本信息

### 世界名稱
- **官方名稱**：知識大陸 (Continentia Sapientiae)
- **別稱**：學之域、智慧洲、知識領地
- **創造時期**：太古紀元（The Primordial Age）
- **當前紀年**：未定紀年（The Age Unchronicled）

### 世界規模
- **面積**：約為地球歐洲大陸 2 倍
- **人口**：數百萬智慧生物
- **時區**：7 個主要時區（對應 7 大王國）
- **語言**：統一語言「知識語」，各地方言各異

## 第五卷 · 架構與系統

### 知識體系分層

```
Tier 1: 基礎知識層
  ├─ 初級概念
  ├─ 基本技能
  └─ 簡單應用

Tier 2: 進階知識層
  ├─ 深入理解
  ├─ 複雜技能
  └─ 實踐應用

Tier 3: 精通知識層
  ├─ 理論掌握
  ├─ 高級技能
  └─ 創新應用

Tier 4: 宗師知識層
  ├─ 完全理解
  ├─ 領域宗師
  └─ 知識創造
```

### 魔法系統與學科對應

```
魔法系統        學科領域        核心概念
─────────────────────────────────────
咒法系    ↔    數學        邏輯、計算、精確
塑能系    ↔    物理        力、運動、能量
鍊金系    ↔    化學        反應、轉化、元素
生命系    ↔    生物        進化、代謝、生命
幻術系    ↔    文學        敘述、表現、想像
時空系    ↔    歷史        時間、因果、記憶
心靈系    ↔    哲學        思想、真理、意識
```

## 第七卷 · 文化與習俗

### 各王國文化特色

**數學王國**
- 崇尚：精確、完美、邏輯
- 節日：「平衡日」- 慶祝秩序與對稱
- 食物：幾何形狀的食物
- 建築：完美對稱的設計

**物理王國**
- 崇尚：力量、運動、實踐
- 節日：「速度節」- 各種競賽
- 食物：高能量食物
- 建築：動力系統驅動的機械

**化學王國**
- 崇尚：變化、反應、轉化
- 節日：「元素節」- 慶祝各種反應
- 食物：精細調配的佳餚
- 建築：利用化學原理的建築

**生物王國**
- 崇尚：自然、平衡、進化
- 節日：「生命日」- 慶祝新生命
- 食物：有機天然食物
- 建築：與自然融為一體

**文學王國**
- 崇尚：美、表達、創意
- 節日：「故事節」- 講述傳奇
- 食物：美麗呈現的藝術餐點
- 建築：充滿藝術感的設計

**歷史王國**
- 崇尚：記憶、因果、傳承
- 節日：「紀年節」- 紀念歷史事件
- 食物：傳統古老食物
- 建築：保留歷史痕跡的建築

**哲學王國**
- 崇尚：思辨、真理、智慧
- 節日：「大辯論」- 進行思想碰撞
- 食物：簡樸素食
- 建築：冥想與思考的空間
