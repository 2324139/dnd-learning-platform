# 🌍 人生即 RPG：歷史編年冒險系統

## 遠古之願景

本系統將凡人的人生轉化為史詩冒險。

**核心理念**：每個玩家皆為「轉生者」，跨越時空，
在真實歷史事件中做出決策，改變自己與世界的命運。

---

## 第一卷 · 時間線架構

### 人生的四個時代

```
Epoch I: 童年時代（Level 1-15）
├─ 古埃及（公元前 3000 年）
├─ 古代中國（東周春秋）
└─ 古希臘（雅典民主時代）

Epoch II: 青年時代（Level 16-40）
├─ 中世紀（公元 500-1400 年）
├─ 伊斯蘭黃金時代（8-13 世紀）
└─ 文藝復興（14-17 世紀）

Epoch III: 壯年時代（Level 41-75）
├─ 大航海時代（15-17 世紀）
├─ 科學革命（16-18 世紀）
└─ 工業革命（18-19 世紀）

Epoch IV: 晚年時代（Level 76-100）
├─ 信息時代（20 世紀）
├─ 太空時代（1960-2000）
└─ 未來時代（2000 年-）
```

## 第二卷 · 核心系統

### 人生屬性

```
STR (力量)       體能、軍事、生存
DEX (敏捷)       藝術、工藝、細節
CON (體質)       心理韌性、壽命、健康
INT (智慧)       知識、創新、邏輯
WIS (感知)       人際、直覺、洞察
CHA (魅力)       領導、說服、影響力
```

### 聲望系統

```
Prestige Tracks (多維度評價)

政治聲望      (-100 ~ +100)    統治者/叛亂者
民眾愛戴度    (-100 ~ +100)    英雄/暴君
學術地位      (-100 ~ +100)    知識傳人/異端
藝術成就      (-100 ~ +100)    大師/無名之輩
道德評價      (-100 ~ +100)    聖人/惡棍
宗教影響力    (-100 ~ +100)    宗教領袖/異教徒
```

### 遺產系統

```
每位玩家都可能留下永久遺產：

✨ 政治遺產：建立制度、改變法律
✨ 文化遺產：著作、藝術、音樂
✨ 科技遺產：發明、理論、技術
✨ 精神遺產：宗教、哲學、思想
✨ 家族遺產：後代、血脈、家風
✨ 建築遺產：城堡、廟宇、紀念碑
```

## 第三卷 · 真實歷史事件

### 示例一：三國時期（184-280 年）

**主線任務樹**
```
黃巾起義（184 年）
├─ 路線 A：加入農民起義
│  ├─ 任務：突破黑山軍防線
│  ├─ NPC：張角、張寶、張梁
│  ├─ 結果：STR +10, INT +5, CHA -5
│  └─ 結局：領袖/戰死/投降
│
└─ 路線 B：投靠官府
   ├─ 任務：鎮壓黃巾
   ├─ NPC：皇帝、郡守、名將
   ├─ 結果：WIS +8, CHA +10, CHR -8
   └─ 結局：功臣/貪官/忠臣
```

**支線任務**
- 拜見諸葛亮學習兵法
- 與關羽、張飛結義
- 參與官渡、赤壁、夷陵三大戰役
- 編纂《三國志》
- 建立蜀、魏、吳聯盟或對抗

**歷史人物（可互動）**
- 諸葛亮：軍事智者
- 曹操：統一者
- 劉備：仁義者
- 孫中山：革新者
- 呂布：戰神

## 第四卷 · 遊戲機制

### 決策系統

```
每個決策都有三層結果：

Level 1：直接結果
「你幫助農民起義」
→ STR +5, INT +8, CHA -5

Level 2：故事結果
「農民起義失敗，你逃亡山林」
→ 下一章的故事改變

Level 3：歷史結果
「你的決策改變了歷史軌跡」
→ 其他玩家的遊戲也受影響
→ 被記入時間線
```

### 分支時間線

```
主時間線（歷史原貌）
├─ 玩家 A 的時間線（保護諸葛亮）
│  └─ 諸葛亮壽命 +20 年
│     └─ 蜀漢國力 +30%
│
├─ 玩家 B 的時間線（暗殺曹操）
│  └─ 三國統一延遲 50 年
│
└─ 玩家 C 的時間線（創立第四方勢力）
   └─ 完全不同的歷史結局
```

### 傳承系統

```
當玩家結束一個時代時，可以：

1️⃣ 啟蒙下一代
   └─ 你的知識傳承給後代角色
   └─ 屬性加成 +10% 每代

2️⃣ 建立學派
   └─ 創立思想或技能系統
   └─ 影響整個文明

3️⃣ 改變技術線
   └─ 提前發明技術
   └─ 加速或延遲歷史

4️⃣ 建立王朝
   └─ 後裔繼承統治
   └─ 建立千年帝國
```

## 第七卷 · 跨時代合作

### 時空同盟系統

```
多個玩家可以跨時代聯繫：

時代 1：玩家 A 完成「造紙術改進」
  ↓
時代 2：玩家 B 發現玩家 A 的遺產
  → 獲得「改進的造紙術」技能
  → 可更快完成「印刷術」發明
  ↓
時代 3：玩家 C 基於前兩者成就
  → 發明「活字印刷術」
  → 整個文明進入新時代

【時空同盟任務示例】
- "保護關鍵歷史人物"：防止某人被暗殺
- "加速技術發展"：多個玩家合作發明
- "建立思想流派"：共同創立新哲學
- "改寫歷史結局"：集合力量改變大事件
```

## 第九卷 · 高級系統

### 時間悖論機制

```
玩家可以故意造成時間悖論：

【悖論示例 1】
你在文藝復興拯救了年輕的達芬奇
→ 他活到 150 歲
→ 發明了飛機（比歷史早 500 年）
→ 時間線分裂
→ 其他玩家在另一條時間線遊玩

【悖論示例 2】
你殺死了嬰兒期的希特勒
→ 二戰未發生
→ 歷史完全改寫
→ 所有玩家進入新的平行宇宙遊戲
```

### 命運對抗系統

```
玩家可以互相對抗（不同時代）：

玩家 A（古代中國）
  └─ 下棋：通過歷史決策間接對抗

玩家 B（文藝復興）
  └─ 競爭：爭奪資源和影響力

玩家 C（現代）
  └─ 評估：被記入史冊的評價競爭

最終結局：看誰的遺產最偉大
```
